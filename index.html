Chocomint CSS

Minimal CSS framework based on Simple CSS
By DiegoBarMor

About

Chocomint CSS is a minimal CSS framework based on Simple CSS, adapted to have a more neon and minty look. High contrast can be achieved with bold text, marked text or code_blocks. Icon buttons offer further contrast and reactivity:

This is a blockquote. It can be used to highlight important information. It can also be used to separate content.

Article blocks are also another way to separate content.

Navigation Bar

Chocomint CSS provides a dynamic navigation bar. When scrolling down, the navigation bar will disappear. When scrolling up, the navigation bar will reappear. The speed at which it does so can be adjusted in chocomint.js (speed variable).

Background Particles

The background particles are achieved by pure CSS, implemented in star-particles.css. An HTML element must be added in the body to achieve this effect. 


          <div class="bg-animation">
            <div id="stars0"></div><div id="stars1"></div>
            <div id="stars2"></div><div id="stars3"></div>
          </div>
